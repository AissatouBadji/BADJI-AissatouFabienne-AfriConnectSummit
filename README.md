## Description du projet

AfriConnect Summit 2026 est un site web vitrine créé pour présenter une conférence tech panafricaine fictive réunissant développeurs, entrepreneurs et investisseurs africains.

Le site présente l'événement, les intervenants, le programme des conférences ainsi qu'un formulaire d'inscription.

L'objectif est de proposer une interface moderne, responsive et interactive adaptée aux événements technologiques.

---

## Technologies utilisées

- HTML5 pour la structure des pages web
- CSS3 pour le design, les animations et le responsive design
- JavaScript Vanilla pour les fonctionnalités interactives
- Google Fonts pour la typographie
- Bootstrap Icons pour les icônes
- Git et GitHub pour la gestion du projet

---

## Pages du site

- *index.html* : page d'accueil avec présentation de l'événement, statistiques, sponsors et intervenants vedettes.
- *programme.html* : présentation du programme sur 3 jours avec des onglets interactifs.
- *intervenants.html* : liste des intervenants avec système de filtrage.
- *contact.html* : formulaire d'inscription, FAQ et localisation.

---

## Fonctionnalités JavaScript

Les fonctionnalités JavaScript développées sont :

- Mode sombre / mode clair avec sauvegarde dans localStorage.
- Menu hamburger pour la navigation mobile.
- Changement dynamique de la navbar au défilement.
- Compte à rebours avant le début de la conférence.
- Animation des statistiques.
- Apparition des sections au scroll avec IntersectionObserver.
- Système d'onglets pour le programme.
- Filtrage dynamique des intervenants.
- Validation du formulaire d'inscription.
- Bouton retour en haut.
- Affichage automatique de l'année dans le footer.
